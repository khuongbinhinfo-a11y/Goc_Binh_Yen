# Gợi ý allowlist cho Codex Cloud

Dùng khi bạn bật internet access cho Codex Web / Cloud.

## Mục tiêu
Cho phép Codex nghiên cứu mood ảnh, hình tham chiếu và bối cảnh công khai cho website Hồn Thơ.

## Domain gợi ý
- google.com
- pinterest.com
- pinimg.com
- youtube.com
- facebook.com
- instagram.com
- tiktok.com
- unsplash.com
- pexels.com
- pixabay.com
- wikimedia.org

## HTTP methods gợi ý
- GET
- HEAD
- OPTIONS

## Ghi chú
- Chỉ nên mở domain thật sự cần.
- Không nên mở POST / PUT / PATCH / DELETE nếu không có lý do rõ ràng.
- Pinterest và mạng xã hội nên xem là nguồn tham chiếu mood, không phải mặc định là nguồn ảnh production.
