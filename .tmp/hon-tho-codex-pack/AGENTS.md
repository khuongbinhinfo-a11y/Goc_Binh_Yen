- Tên tác giả phải nằm **dưới bài**

  - Liên hệ

## Quy tắc web research
Khi cần tìm mood ảnh hoặc nguồn tham chiếu:
1. Dùng web search trước
2. Có thể khảo sát nguồn công khai như Pinterest, YouTube, Facebook page công khai, Instagram web công khai, TikTok web công khai
3. Tách rõ:
   - mood reference
   - production candidate
4. Không được mặc định coi Pinterest / mạng xã hội là ảnh có thể dùng trực tiếp cho production
5. Với mỗi bài thơ, lưu lại:
   - keyword tìm kiếm
   - reference links
   - cảnh báo bản quyền nếu có
